# Vinharia Agnello - Projeto Checkpoint de Web Development
 
## Descrição
Sistema básico para cadastro e análise de vinhos artesanais da Vinheira Agnello. O projeto visa facilitar o gerenciamento da produção, permitindo o registro detalhado de cada lote e a análise técnica das características dos vinhos produzidos.
